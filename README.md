# Laravel 11 with AdminLTE

## Installation

1. clone this repository
   ```sh
   git clone https://github.com/defrindr/laravel11-adminlte.git
   ```
2. copy .env.example to .env
2. install dependecies
   ```sh
   composer install
   npm install
   ```
3. run vite
   ```sh
   npm run dev
   ```
4. migration database
   ```sh
   php artisan migrate --seed
   ```
4. run artisan serve
   ```sh
   php artisan serve
   ```
